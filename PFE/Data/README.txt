This folder originally contained the datasets used in this project.

- Breast Cancer Wisconsin (Diagnostic) Dataset (WDBC):
  Available on Kaggle: https://www.kaggle.com/datasets/uciml/breast-cancer-wisconsin-data

- COVID-19 Radiography Database:
  Available on Kaggle: https://www.kaggle.com/datasets/tawsifurrahman/covid19-radiography-database

The datasets were used respectively for:
- MLP experiments (Breast Cancer classification)
- CNN experiments (COVID-19 detection from chest X-ray images)

Please download them manually if you want to reproduce the experiments.
